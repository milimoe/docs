Markdown 示例

Markdown 是一种轻量级的标记语言，用于以纯文本格式编写格式化的文档。以下是 Markdown 的一些常用语法：

标题

使用 # 来表示标题，# 的数量表示标题的级别，1 到 6 级：

# 一级标题
## 二级标题
### 三级标题
#### 四级标题
##### 五级标题
###### 六级标题

粗体和斜体

使用 ** 或 __ 表示粗体，使用 * 或 _ 表示斜体：

这是 **粗体** 文字
这是 __粗体__ 文字
这是 *斜体* 文字
这是 _斜体_ 文字

列表

无序列表使用 -、* 或 +，有序列表使用数字加点：

无序列表：
- 项目 1
- 项目 2
  - 子项目 1
  - 子项目 2

有序列表：
1. 项目 1
2. 项目 2
   1. 子项目 1
   2. 子项目 2

链接和图片

链接使用 [描述](URL)，图片使用 ![描述](URL)：

这是一个 [链接](http://example.com)
![示例图片](http://example.com/image.jpg)

引用

使用 > 表示引用：

> 这是一个引用

代码和代码块

单行代码使用反引号（`），代码块使用三个反引号（```）：

这是 `inline code`

这是一个代码块：

分割线

使用三个或更多的 *、- 或 _：

---

表格

使用 | 创建表格：

| 头1 | 头2 | 头3 |
| --- | --- | --- |
| 单元格1 | 单元格2 | 单元格3 |
| 单元格4 | 单元格5 | 单元格6 |

其他

  • 删除线 使用 ~~ 包围文本：

    ~~这是一段删除的文字~~

  • 任务列表 使用 [ ] 和 [x]：

    - [ ] 未完成任务
- [x] 已完成任务

以上是 Markdown 的一些常用语法，可以帮助你编写格式化的文档。
